# NetErrror Wallet Scanner | RU 🇷🇺

Многопоточный/многопроцессный генератор BIP39-мнемоник с проверкой баланса BTC и ETH.  
Сканирует случайные 12-словные фразы, конвертирует их в ключи и проверяет баланс через публичные API.  
**Только для образовательных целей!**

## Возможности

- Генерация мнемоник BIP39 (12 слов) с использованием криптографически стойкого `os.urandom`
- Конвертация мнемоники → сид → мастер-приватный ключ (BIP32) → адреса BTC (P2PKH) и ETH (EIP-55)
- Проверка баланса через публичные API:
  - Bitcoin: `blockchain.info/balance`
  - Ethereum: `etherscan.io/api` (без API-ключа, анонимно)
- Многопроцессорность + многопоточность для высокой скорости (настраивается)
- Автосохранение найденных кошельков в `found_wallets.json`
- Прогресс-бар в реальном времени, статистика по найденным

## Требования

- Python 3.8+
- Стандартная библиотека (без внешних зависимостей)
